# Autenticacao-e-Validacao-Web-em-JS
Exemplo de autenticação e validação entre cliente/servidor via JS, utilizando a ferramenta Ngrok para simular um servidor na internet e validando a autenticação do login e senha via BasicAuth.

OBS: Antes de tudo instale as seguintes dependências antes de rodar o projeto
> npm init -y

> npm install express
